# JavaWebProject

### JavaWeb 学习笔记
一个比较经典的JavaWeb简单的练习了，留言板功能，之前初次学习的时候做过，现在重新捡起来，重写了一遍，主要是复习了一下基础。Servlet，filter，session和jstl，EL表达式等。

功能包括有注册登录，查看和修改个人信息，留言以及查看自己的留言。
